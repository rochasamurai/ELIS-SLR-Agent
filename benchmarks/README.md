# Benchmarks

This folder contains benchmark configs, queries, scripts, fixtures, and generated outputs.

## Layout
- `benchmarks/config/` benchmark configuration
- `benchmarks/queries/` benchmark queries
- `benchmarks/scripts/` benchmark runners and adapters
- `benchmarks/fixtures/` small deterministic inputs/expected outputs
- `benchmarks/outputs/` generated outputs (ignored)
- `benchmarks/reports/` generated reports (ignored)
