# ELIS Benchmark 1: Darmawan (2021)

**Benchmark ID:** DARMAWAN_2021  
**Status:** COMPLETED  
**Executed:** January 2026

## Overview

This benchmark validated ELIS SLR Agent against a semi-systematic review on e-voting adoption.

**Paper:**
Darmawan, I. (2021). E-voting adoption in many countries: A literature review. *Asian Journal of Comparative Politics, 6*(4), 482-504.

**Key Details:**
- **Studies:** 78 articles (2005-2020)
- **Databases:** Google Scholar, ACM Digital Library, Science Direct, JSTOR
- **Focus:** E-voting adoption research

## Results

| Metric | Value |
|--------|-------|
| **Retrieval Rate** | 66.67% (52/78) |
| **Precision** | High |
| **Cost** | \$0.50 |
| **Runtime** | 22 minutes |
